## 第一阶段：前置理论与核心技能储备


### 1.1 核心基础理论学习

  

1. **小行星光变测自转周期的核心原理**

  

    - 核心逻辑：不规则形状的小行星自转时，朝向地球的反射面积/反照率发生周期性变化，产生**光变曲线**（亮度随时间的变化曲线），光变曲线的稳定周期即为小行星的自转周期。

  

    - 必学概念：视星等/仪器星等/绝对星等、相位角、光变幅度、大气消光、光行差、日心儒略日（HJD）、信噪比（SNR）、半高全宽（FWHM）。
	- 仪器星等 ：m<sub>i</sub> =-2. 5lg(D<sub>is</sub> - D<sub>ib</sub>) +c
		- m<sub>i</sub> 为i系统的颜色下的星等
		- D<sub>is</sub> 为被测星的读数
		- D<sub>ib</sub> 为天光背景的读数
	- 相位角 ：太阳-小行星-地球的夹角
	- 光变幅度 ：最亮与最暗时的星等之差
	- 日心儒略历 ：消除光行差的影响
	- 信噪比 ：实测天体物理3.2.4
  

    - 关键常识：绝大多数小行星的光变曲线为**双峰结构**（自转一周出现2个极大/极小值），自转周期集中在2-24小时，主带小行星周期多为4-12小时，近地小行星可能出现超短周期（<2小时）。

  

2. **天文测光的核心方法**

  

    - 区分**相对测光（差分测光）与绝对测光：小行星自转周期测量仅需相对测光**（消除系统误差，保留亮度的相对变化），无需绝对定标，大幅降低入门难度。

  

    - 必学核心技术：**集合法差分测光（Ensemble Differential Photometry）**，使用视场内多颗稳定比较星的平均流量作为参考，消除大气消光、仪器响应、曝光波动等系统误差，是小行星测光的行业标准。

  

- 掌握两种测光方式的适用场景：
	- **孔径测光**：视宁度好、星点不拥挤的场景，新手优先上手，操作简单、鲁棒性强；
		- 画个圈，把圈里的光全加起来
	- PSF测光：视宁度差、星点拥挤、目标暗弱的场景，测光精度更高，进阶使用。
		- 先拟合星点形状，再数学计算总流量

## 第二阶段：目标小行星的筛选与确定（调研核心产出）

  

本阶段是项目成败的关键，新手优先选择**高可观测性、高信噪比、低难度**的目标，先完成方法验证，再挑战未知/暗弱目标，建议周期3-7天。

  

### 2.1 明确筛选优先级

  

|筛选维度|新手推荐阈值|核心逻辑与避坑提示|
|---|---|---|
|目视星等|亮于14.5等（优先**10-13**等）|星等每暗1等，流量下降2.5倍，14.5等是20-40cm口径望远镜的入门极限；亮于10等的目标多已被多次测量，适合练手验证|
|光变幅度|≥0.3mag（最低≥0.1mag）|光变幅度必须大于测光精度的10倍（如测光精度0.03mag，幅度需≥0.3mag），否则噪声会完全淹没光变信号；不规则小行星幅度更大，更适合新手|
|自转周期|2-12小时|一整夜可覆盖1-2个完整周期，无需跨夜观测，避免相位衔接、大气条件变化带来的误差；<2小时的超短周期需要极高采样频率，>12小时需要跨夜，新手暂不考虑|
|观测窗口|冲日前后1个月内|冲日时小行星相位角接近0°，亮度最高、相位效应最小，是最佳观测窗口；相位角建议≤30°，避免相位暗化、散射光影响光变曲线|
|赤纬匹配|与观测台站纬度差≤30°|保证观测时目标地平高度≥30°（优先≥45°），地平高度过低会导致大气消光急剧增大，测光误差飙升（如北京地区优先赤纬-10°~+70°）|
|自行|曝光时间内移动≤1个像素|避免长曝光导致小行星拖线，拖线会严重降低测光精度；自行过大的近地小行星，需缩短曝光时间，新手优先主带小行星（自行小、轨道稳定）|
|已有测量结果|LCDB中U=1~2（不确定结果）或U=3（验证用）|LCDB的质量码U=3为可靠结果，适合新手练手验证方法；U=1~2为不确定/低精度结果，有科研创新性，适合正式项目|

### 2.2 分步执行目标筛选

  

1. **打开核心数据库**：登录[LCDB小行星光变数据库](https://sbn.psi.edu/pds/resource/lc.html)、JPL Horizons星历系统、MPC小行星数据库；

  

2. **初筛设置**：在LCDB中按上述阈值设置筛选条件，导出符合要求的小行星列表，优先排除已多次高精度测量（U=3）的目标（练手除外）；

  

3. **观测窗口核对**：将初筛目标输入JPL Horizons，输入你的观测地点、观测时间段，核对以下参数：

  

    - 观测时段内目标的地平高度≥30°的连续时长≥2倍预估自转周期；

  

    - 月相与目标的角距离≥60°，月相≤30%（无月夜最佳），避免月光污染；

  

    - 目标的自行大小，计算最大允许曝光时间；

  

4. **视场与比较星验证**：

  

    - 下载目标天区的Digitized Sky Survey（DSS）图像，确认视场内有≥5颗与目标亮度差≤2mag的恒星；

  

    - 通过Gaia DR3星表验证这些恒星的稳定性：排除变星、高自行天体、双星，确保比较星的光变稳定性优于0.01mag；

  

5. **最终目标确定**：

  

    - 新手优先确定1颗**已知周期的亮目标**（如1号谷神星、4号灶神星，光变幅度稳定、周期已知）作为方法验证目标；

  

    - 同时确定1-2颗**科研目标**（LCDB低质量码、无高精度周期测量）作为正式项目目标；

  

    - 备份2-3颗备选目标，应对天气、观测条件变化。

## 第三阶段：观测方案设计与设备准备（调研落地阶段）

  

本阶段目标是制定可执行的观测计划，确保观测数据满足测光精度与周期计算的要求，建议周期3-7天。

  

### 3.1 观测设备选型与参数确认

  

1. **望远镜系统**

  

    - 最低要求：口径≥20cm的施密特-卡塞格林（SC）/牛顿反射望远镜，焦比F/5~F/10；

  

    - 核心要求：光学系统稳定、无明显像差，配备赤道仪（必须带导星功能，优先离轴导星），赤道仪跟踪精度≤1角秒/30分钟；

  

2. **成像相机**

  

    - 必须使用**制冷CCD/CMOS相机**，制冷温度低于环境温度30℃以上，抑制热噪声；

  

    - 像元尺度：匹配观测地视宁度，最终像元尺度12角秒/像素（最佳为视宁度的1/21/3）；

  

    - 避免使用无制冷的单反/微单相机，热噪声会严重降低测光精度。

  

3. **滤光片选择**

  

    - 新手优先使用**Johnson-Cousins R波段**：受大气消光影响小、月光污染低于B/V波段、小行星在红光波段流量更高，信噪比更好；

  

    - 禁止无滤光片观测：无滤光片的宽带测光会受色差、大气消光影响，系统误差极大，无法得到可靠光变曲线；

  

4. **导星系统**：必须配备导星镜/离轴导星器+导星相机，保证长曝光时星点FWHM稳定，无拖线、无漂移。

  

### 3.2 观测时序与曝光方案设计

  

1. **曝光时间计算**

  

    - 核心目标：目标小行星的单帧信噪比SNR≥100（最佳≥200），对应测光精度≤0.01mag；

  

    - 上限约束：单帧曝光时间内，小行星的自行移动≤1个像素，避免拖线；

  

    - 测试方法：先拍摄5-10张测试帧，调整曝光时间，确保目标星点不饱和、FWHM稳定、背景噪声均匀。

  

2. **采样间隔设计**

  

    - 遵循奈奎斯特采样定理：采样频率≥2倍的最高光变频率，即采样间隔≤预估自转周期的1/4；

  

    - 新手推荐：预估周期6小时，采样间隔5-10分钟，保证一个周期内有≥36个采样点，避免出现假周期；

  

    - 禁止长时间中断观测：连续观测的中断时长≤30分钟，避免周期搜索出现歧义。

  

3. **总观测时长要求**

  

    - 必须覆盖**至少2个完整的自转周期**，如预估周期6小时，总连续观测时长≥12小时；

  

    - 条件允许时，分2个不同夜晚观测，用于交叉验证周期的可靠性。

  

4. **校准帧拍摄计划（必做，否则数据作废）**

  

    - 偏置帧（Bias）：曝光时间0s，相同相机设置，拍摄≥20张，用于消除相机本底噪声；

  

    - 暗场帧（Dark）：曝光时间与科学帧完全一致，相同制冷温度，拍摄≥15张，用于消除热噪声；

  

    - 平场帧（Flat）：相同滤光片、相同相机设置，拍摄≥20张，用于消除像素响应不均匀、光学渐晕；推荐在晨昏时段拍摄天光平场（太阳在地平线下10-15°，天光均匀无星点）。

  

### 3.3 观测前的预准备

  

1. 编写详细的观测脚本，包含目标坐标、观测时段、曝光参数、采样间隔、校准帧拍摄流程；

  

2. 提前下载目标的高精度星历（JPL Horizons生成，时间精度到分钟），导入观测软件，确保能自动追踪小行星；

  

3. 完成赤道仪极轴校准，极轴精度≤30角秒，保证长时间跟踪的稳定性；

  

4. 测试导星系统，确保导星精度稳定在1角秒以内。

## 第四阶段：观测执行与数据质量管控

  

本阶段目标是获取高质量的原始数据，避免无效观测，核心是实时监控数据质量。

  

1. **观测前检查**：确认极轴校准、导星系统、相机制冷温度稳定，完成校准帧拍摄（平场帧优先晨昏拍摄）；

  

2. **试观测**：拍摄5-10帧测试图像，实时检查：

  

    - 小行星在视场内，无拖线、无饱和；

  

    - 星点FWHM稳定，视宁度≤2角秒；

  

    - 比较星全部在视场内，无截断、无拥挤；

  

3. **连续观测执行**：严格按照观测脚本执行，实时监控导星状态、天气变化、云量遮挡；

  

4. **实时质量管控**：每30分钟检查一次图像，若出现云遮挡、导星丢失、星点拖线、月光闯入，立即暂停观测，剔除坏帧，待条件恢复后继续；

  

5. **观测收尾**：观测结束后，立即拍摄暗场帧、偏置帧，完整备份所有原始数据（科学帧+校准帧），做好数据命名与记录。

## 第五阶段：数据预处理与测光处理（核心计算环节）

  

本阶段目标是从原始图像中提取高质量的小行星光变曲线，是周期计算的基础，建议周期1-2周。

  

### 5.1 原始数据预处理

  

1. **校准帧合并**：使用IRAF/MaxIm DL/Python，对偏置帧、暗场帧、平场帧做中值合并，生成主偏置、主暗场、主平场；

  

2. **科学帧校准**：对每一张科学帧执行校准流程：`校准后图像 = (原始图像 - 主偏置 - 主暗场) / 主平场`，消除系统误差；

  

3. **坏像素/宇宙线剔除**：使用L.A.Cosmic算法剔除宇宙线，修复坏像素，避免影响测光精度；

  

4. **天文定标与坐标校正**

  

    - 使用astrometry.net对每一张校准后的图像做盲解，匹配UCAC4/Gaia DR3星表，得到图像的天球坐标解算结果（WCS）；

  

    - 导入小行星的高精度星历，在每一张图像中精准定位小行星的位置（小行星有自行，每帧位置不同，禁止手动定位）；

  

    - 剔除小行星位置超出视场、导星误差过大的坏帧。

  

### 5.2 差分测光核心流程

  

1. **比较星最终确认**

  

    - 在所有图像中，提取视场内预选的5-10颗比较星的流量数据；

  

    - 计算每颗比较星的相对光变，剔除光变幅度>0.02mag的不稳定恒星/变星，最终保留≥3颗稳定的比较星；

  

2. **测光执行**

  

    - 测量每帧图像中，小行星与所有比较星的FWHM，设置统一的测光孔径（推荐为平均FWHM的2-3倍）；

  

    - 设置天空背景环：内径为平均FWHM的5倍，外径为8倍，确保背景环内无其他星点、无饱和区域，精准计算天空背景流量；

  

    - 对小行星和所有比较星执行孔径测光，得到每颗天体的**净流量**（总流量 - 天空背景流量）、测光误差；

  

3. **差分星等计算**

  

    - 计算每颗比较星的仪器星等：`仪器星等 = -2.5 * log10(净流量 / 曝光时间)`；

  

    - 对所有比较星的仪器星等做加权平均，得到参考星等；

  

    - 计算小行星的差分星等：`差分星等 = 小行星仪器星等 - 参考星等`；

  

    - 输出核心结果：**时间序列表**，包含每帧的HJD时间、小行星差分星等、测光误差。

  

4. **数据质量筛选**

  

    - 剔除测光误差>0.05mag的低质量数据点；

  

    - 剔除云遮挡、信噪比过低的异常点；

  

    - 最终得到信噪比均匀、无系统趋势的光变曲线。

## 第六阶段：自转周期计算与可靠性验证（核心成果输出）

  

本阶段目标是从光变曲线中解算出可靠的自转周期，并完成多维度验证，确保结果的科学性，建议周期1周。

  

### 6.1 时间序列预处理

  

1. **时间校正（必做，否则周期结果无效）**

  

    - 将所有观测时间从UTC转换为**日心儒略日（HJD）**，校正地球公转带来的光行差效应，消除时间系统误差；

  

    - 禁止直接使用UTC/观测地时间做周期计算，光行差带来的时间偏差可达几分钟到几十分钟，会严重影响短周期的相位精度。

  

2. **数据归一化**：对差分星等做归一化处理，消除长期系统趋势（如大气消光的慢变化）。

  

### 6.2 周期搜索与解算

  

1. **Lomb-Scargle周期图计算**

  

    - 设置合理的频率搜索范围：对应周期0.5小时~48小时，覆盖预估周期的±50%；

  

    - 设置足够高的频率分辨率：频率步长≤1/(10*总观测时长)，保证周期的计算精度；

  

    - 计算周期图，找到显著性最高的峰值，对应的周期为初步候选周期；

  

2. **虚警概率（FAP）检验**

  

    - 计算候选周期的FAP值，**只有FAP<1%的周期才是统计显著的**，排除噪声带来的假周期；

  

    - 若FAP>5%，说明候选周期不可靠，需重新检查光变曲线、补充观测数据。

  

3. **相位折叠与光变曲线优化**

  

    - 用候选周期对光变曲线做相位折叠，将时间转换为0-1的相位值，绘制折叠后的光变曲线；

  

    - 检查折叠后的光变曲线：是否呈现光滑的周期性变化（多为双峰结构），极大/极小值是否重复，散点是否集中；

  

    - 微调周期值，使相位折叠后的光变曲线色散最小，得到最优周期。

  

### 6.3 多维度周期验证（必做，科研规范要求）

  

1. **算法交叉验证**：使用PDM、MCMC等方法重新计算周期，确认与Lomb-Scargle的结果一致，偏差在误差范围内；

  

2. **误差估计**：使用bootstrap蒙特卡洛方法，对光变曲线做重采样，计算周期的1σ误差，规范输出格式为`P = X.XXX ± 0.00X 小时`；

  

3. **分段验证**：将观测数据分为前后两段，分别计算周期，确认两段的周期一致，无显著偏差；

  

4. **跨夜数据验证**：若有多个夜晚的观测数据，确认不同夜晚的光变曲线相位能完美衔接，周期一致；

  

5. **文献比对**：与LCDB数据库中已有的测量结果比对，若为已知周期目标，确认结果在误差范围内一致，验证方法的可靠性。

## 第七阶段：结果分析与成果整理

  

1. **结果分析**

  

    - 分析光变曲线的形状、幅度，推断小行星的形状特征（光变幅度越大，形状越不规则）；

  

    - 分析相位角、观测条件对结果的影响，讨论与已有结果的差异原因；

  

    - 排查是否存在伴星/双星系统（光变曲线出现掩食/凌星信号）、非-principal axis rotation（非主轴自转）等特殊情况。

  

2. **成果整理**

  

    - 按照天文期刊规范，整理完整的观测日志、数据处理流程、周期结果、光变曲线、误差分析；

  

    - 若为新的高精度周期测量结果，可提交至LCDB数据库更新，或撰写论文投稿至相关期刊；

  

    - 归档所有原始数据、处理代码、结果文件，保证科研结果的可复现性。

## 新手避坑核心提示

  

1. 先练手再创新：先用已知周期的亮目标完成全流程验证，确认方法、设备、处理流程无误后，再挑战未知目标；

  

2. 精度优先：测光精度是周期计算的基础，所有环节都要围绕“降低测光误差”展开，不要盲目追求暗弱目标；

  

3. 拒绝假周期：必须完成FAP检验和多维度交叉验证，不要把周期图的噪声峰值当成真实周期；

  

4. 规范时间系统：必须使用HJD日心儒略日，光行差校正绝对不能省略。
