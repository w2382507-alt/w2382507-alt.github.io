# 文本格式
## 段落

换行是使用两个空格加回车   
或者使用空行表示开始一个段落

## 字体

斜体：一个*
粗体：两个**
粗斜体：三个***
这段文字包含**粗体**、*斜体*和***粗斜体***的组合效果。

## 分隔线

可以在一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西。你也可以在星号或是减号中间插入空格

## 删除线

如果段落上的文字要添加删除线，只需要在~~文字~~的两端加上两个波浪线 ~~ 即可

## 下划线

<u>带下划线文本</u>

## 脚注

创建脚注格式类似这样 [^RUNOOB]。

[^RUNOOB]: 菜鸟教程 -- 学的不仅是技术，更是梦想！！！

## 文本高亮

这是==文本==高亮


# 列表

## 无序列表

无序列表使用星号( * )、加号( + )或是**减号\( \- \)** 作为列表标记，这些标记后面要添加一个空格，然后再填写内容

## 有序列表

1. 第一项
2. 第二项
3. 第三项

- 注意要加空格
- 有序列表和无序列表都可以嵌套，保持**2个空格缩进**

## 任务列表

- [ ] 未完成的任务
- [x] 已完成的任务
- [ ] 另一个未完成的任务
- [x] 你好
